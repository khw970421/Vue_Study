## 필수로 하는것 

npm init -y<br>
npm i vue@next<br>
npm i -D parcel<br>
npm i -D sass<br>

## 선택으로 하는 것 
npm run dev<br>
npm run build

### npm run dev
>개발용도로 개발서버가 오픈

### npm run build
>build는 배포용도로 개발서버가 열리지않는다.<br>
 대신 코드가 가독성이 좋은 코드가 아닌 암호화(?) 비슷하게 되어있어서 dev에서 하던 필요없는 주석과 코드를 줄여준다.

* npm run build 한 dist폴더의 index.html을 크롬 브라우저 클릭하면 결과가 잘 나오지는 않는다. <br>
* 오히려 npm run dev는 원하는 결과가 잘 나온다.
